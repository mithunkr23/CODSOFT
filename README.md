# Student Management System (Java)

A simple **console-based Student Management System** built using Java. This project allows users to manage student records efficiently through basic operations like adding, removing, searching, and displaying students.

---

## Features

* Add new student
* Remove existing student
* Search student by roll number
* Display all students
* Menu-driven console interface

---

## Technologies Used

* Java
* ArrayList (Java Collections Framework)
* OOP Concepts

---

## Project Structure

```
├── Main.java                     # Entry point of the program
├── Student.java                 # Student class (model)
├── StudentManagementSystem.java # Core logic and operations
```

---

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/codsoft.git
```

### 2. Navigate to Project Folder

```bash
cd codsoft
```

### 3. Compile the Program

```bash
javac Main.java Student.java StudentManagementSystem.java
```

### 4. Run the Program

```bash
java Main
```

---

## Usage

After running the program, you will see a menu like:

```
===== Student Management System =====
1. Add Student
2. Remove Student
3. Search Student
4. Display All Students
5. Exit
```

Enter your choice and follow the prompts.
